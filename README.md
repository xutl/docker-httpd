## Filesystem layout

Directory                       | Description
------------------------------- | ------------------------------------------------------------------------------
`/usr/local/apache2`     | Apache path


File                                                | Description
--------------------------------------------------- | ------------------------------------------------------------------------------
`/usr/local/apache2/conf/httpd.conf`         | Apache configuration
